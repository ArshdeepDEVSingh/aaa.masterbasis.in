Linear Equation in One Variable

  • Most important thing in any equation is that L.H.S. is always equal to R.H.S.
  • When any operation are transferred from L.H.S. to R.H.S. OR R.H.S. to L.H.S., then the operations will be reversed
  • Like this: + → -
  • Like this: - → +
  • Like this: * → ÷
  • Like this: ÷ → *

Example:-

Example 1: 2x-3=7

Solution:

2x=7-3

x=4 ÷ 2

x=2

Example 2:2y+9=4

Solution:

2y=4-9

y=-5/2
